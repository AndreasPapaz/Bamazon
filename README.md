# Bamazon
------

### amazon like store front node app with sql database


# Description
------

Using mysql i created a store front that manages products through the node.js app interface. There are three **interface modes** for this app.

* the customer view
* the manager view
* the supervisor view


# Technologies
---

* node.js
* mysql
* console.table
* inquirer
